#### 주소와 이름

+ IP 주소

  + IPv4 프로토콜에서 사용
  + 32비트 크기의 주소 체계
  + IPv6에서는 128 비트 주소 체계로 확장
  + 예 : 211.223.201.30

  

### DNS(Domain Name Server)

상위 계층에서 주소를 요청하면 DNS에서 ip주소로 변환하여보여준다.



+ #### 호스트 이름

  + <호스트> <단체이름> <단체 종류> < 국가 도메인>
  + 예 : zebra.korea.co.kr

  

#### 국가 도메인

| 국가도매인 | 해당 국가명 |
| :--------- | ----------- |
| kr         | 한국        |
| jp         | 일본        |
| us         | 미국        |



#### 단체 종류

| 단체 종류     | 기관 성격      |
| ------------- | -------------- |
| com[Company]  | 회사           |
| ac[Academy]   | 교육기관       |
| go[goverment] | 정부 소속 기관 |
