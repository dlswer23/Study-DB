# Ubuntu의 bash 셸

+ 기본 셸은 bash(Bourne Again Shell : '배시 셸')



+ ##### Bash 셸의 특징 

  + Alias 기능(명령어 단축 기능)
  + History 기능(위/아래 화살표키)
  + 연산 기능
  + Job Control 기능
  + 자동 이름 완성 기능(탭키)
  + 프롬프트 제어 기능
  + 명령 편집 기능

  

+ ##### 셸의 명령문 처리 방법

  + (프롬프트) 명령어 (옵션 | 인자)
  + 예 ) # rm -rf/mydir



### 셸 스크립트 프로그래밍



+ C언어와 유사하게 프로그래밍이 가능
+ 변수, 반복문 , 제어문 드의 사용이 가능
+ 별도로 컴파일하지 않고 텍스트 파일 형태로 바로 실행 
+ vi 나 getdt으로 작성이 가능
+ 리눅스의 많은 부분이 셸 스크립트로 되어 있음



### 파라미터(parameter) 변수

+ 파라미터 변수는 $0,$1,$2 ..의 형태를 가짐
+ 전체 파라미터는 $*로 표현 

예 )

| 명령          | Apt-get | -y   | install | Gift |
| ------------- | ------- | ---- | ------- | ---- |
| 파라미터 변수 | $0      | $1   | $2      | $3   |
